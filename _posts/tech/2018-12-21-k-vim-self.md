> [k-vim](https://github.com/wklken/k-vim) 是 [wklken](http://www.wklken.me/) 基于 vim 安装定制了各种插件的一个结合, 非常好用, 非常推荐, 个人使用 k-vim 已经超过 5 年一上了 :D
> 具体使用请看 github 上的文档等, 这里主要记录我自己使用过程中经常忘掉的快捷键

### 快捷键

```
nerdtree:
# 开启目录树:
r - 刷新当前目录
R - 熟悉根目录树
I - 隐藏文件

编辑过程中熟悉文件
:e - 刷新当前文件

,m 操作菜单
```
